## Project setup
### Step 1
```
git clone https://github.com/vo-dinh-tu/backend_order_food_and_drink.git
```
or
```
git clone git@github.com:vo-dinh-tu/backend_order_food_and_drink.git
```
### Step 2
```
make init
```

## Run Server
```
make run-server
```

## Run Client
```
make run-client
```

## Pull code Server
```
make pull-server
```
